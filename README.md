# Data Structures and Algorithms

A personal collection of DSA implementations built for practice and interview preparation.
Written in Python.

## What's in here

**Data Structures**
- Arrays, Linked Lists, Stacks, Queues
- Trees (Binary Tree, BST, etc.)
- Graphs, Heaps, Hash Tables

**Algorithms**
- Searching — Binary Search, BFS, DFS
- Recursion & Backtracking

## Why this repo

I use this as a living reference while learning new concepts and preparing for
technical interviews. Each implementation is written from scratch to build
genuine understanding rather than just memorizing solutions.

## Languages & Tools

- Editor: VS Code
